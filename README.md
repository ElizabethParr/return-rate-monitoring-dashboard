# Return Rate Monitoring Dashboard

## Project Overview
This project analyzes customer return behavior to identify patterns and potential root causes. Using Tableau, the analysis explores return rates across time, geography, product categories, and customers to understand where returns are concentrated.

The result is an interactive dashboard designed to support ongoing monitoring and root-cause investigation.

## Key Insights
- Return rates vary over time, indicating seasonal effects  
- Return behavior differs by product category and sub-category  
- Some sub-categories show elevated return rates only in specific regions  
- Geographic analysis highlights states with consistently higher return rates  
- A small subset of repeat customers exhibits consistently high return rates  

## Dashboard Features
The **Return Rate Monitor Dashboard** enables users to:
- View the overall return rate as a KPI  
- Analyze return trends over time  
- Compare return rates across product categories and sub-categories  
- Identify geographic concentrations of returns  
- Use interactive filters to isolate potential drivers  

## Revisions
- Revised the *Return Rate by Sub-Category and Region* view from a stacked bar chart to a highlight table based on instructor feedback, improving interpretability of percentage-based return rates.
